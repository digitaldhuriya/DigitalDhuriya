# Business OS Frontend (Next.js)
Digital Dhuriya Business OS admin dashboard frontend.

## Run
```bash
npm run dev --workspace business-os
```

## Build
```bash
npm run build --workspace business-os
```

## Environment
Use `apps/business-os/.env.example` as template.
